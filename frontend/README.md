# Polaflix Frontend

Este directorio contiene un frontend Angular mínimo para el sistema Polaflix.

## Qué incluye
- Home/entrada de la aplicación
- Catálogo de series con botón "Agregar a pendientes"
- Router SPA con parámetros de URL
- Servicio para consumir la API REST del backend

## Cómo usarlo
1. Instala Node.js y npm.
2. En este directorio, ejecuta `npm install`.
3. Ejecuta `npm start`.
4. Asegúrate de que el backend Spring Boot esté activo en `http://localhost:8080`.

## Nota
El backend también incluye una configuración CORS para permitir llamadas desde el frontend local.
